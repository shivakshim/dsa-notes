#### ⌛Brute Force 

- Sort (nlogn) -> calculate.

#### 🗝️Key Concept

- Max_heap -> pop in desc order -> keep calculating diff and pushing till we reach to 1 final value.

### 🧠Intuition/Approach

- Take a max heap.
- push all elements into it.
- start popping 1 by 1 till the pq size is > 1.
- in each loop -> pop first, pop second , find diff.
- if difference > 0, then push again.
- In the end return the top if it exists, otherwise return 0.

### ❓Note

- There can be a case where we have 2 equal elements in the pq, ans we will pop them both out and wont push since diff = 0, in that case the pq will be empty in the end and we will have to return 0.

### ⌛TC 
- Brute - O(n2)
- Optimized - O(nlogn)

### 📦SC
- Brute - O(n)
- Optimized - O(n)

---

### ✅Solution

```cpp
int lastStoneWeight(vector<int>& stones) {
        priority_queue<int> pq;  //declaring a max heap
        for(int i=0; i<stones.size(); i++){
            pq.push(stones[i]);
        }
        while(pq.size() > 1){
            int first = pq.top();
            pq.pop();
            int second = pq.top();
            pq.pop();
            if(first - second > 0) pq.push(first - second);
        }
        return (!pq.empty() ? pq.top() : 0);
    }
```

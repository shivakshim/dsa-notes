#### ⌛Brute Force 

- Push it all into pq, then simply pop and make list.
- Inefficient because it makes pq have all elements leading to n*k size, while we can easily do it by just keeping the current heads to be compared in the pq, this making pq size = k.*

#### 🗝️Key Concept

- Since I can have Multiple lists, its inefficient if I : 
1. Try to put all elements into an array then and sort because the space required will be huge.
2. Use pointers, to scan smallest head and move, because every time scanning takes n (for loop  of n).
- Hence we use PQ -> min heap, that will only have the heads to be compared, thus keeping the size k, also we wont have to scan through all k lists heads(no for loop), we can get the smallest directly by PQ. 
- Since we keep {val, address}, we can simply pop smallest val node, and push its next node for the next comparison in the pq.
- This makes sure we only have k elements in the pq (thus takes care of size) and also we don't have to scan through all (thus takes care of the time).

### 🧠Intuition/Approach

- Push the head of each list into a **min-heap** (key = node value).
- While heap not empty
    - Pop smallest node.
    - Append it to your result list.
    - If that node has a next, push it into the heap.
- Return merged list’s head.

### ❓Note

- We need this because there are K (unknown number of lists), so scaling shouldn't get too inefficient,
- If we had just 2-3 lists we could have used pointers or array.

### 🧩 **TC & SC:**

- ⏱️ **Time:** `O(N log k)` → N = total nodes, k = number of lists.
- 💾 **Space:** `O(k)` for the heap.

---

### ✅Solution

```cpp
ListNode* mergeKLists(vector<ListNode*>& lists) {
        priority_queue<pair<int, ListNode*>, vector<pair<int, ListNode*>> , 
        greater<pair<int, ListNode*>>> pq; //minheap

        for(int i=0; i<lists.size(); i++){
            ListNode *head = lists[i];
            if(head){
                pq.push({head->val, head});
            }
        }

        if(pq.empty()) return nullptr;

        ListNode *ans = nullptr, *p = nullptr;
   
        while(!pq.empty()){
            ListNode *q = pq.top().second; pq.pop();
            ListNode *node = new ListNode(q->val);
            if(!p) p = node;
            else {
                p -> next = node;
                p = node;
            }
            if(!ans) ans = p;

            if(q->next) pq.push({q->next->val, q->next});
        }
        
        return ans;
    }
```

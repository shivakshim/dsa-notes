#### 🗝️Key Concept

- **Do a normal BFS, store each level separately, and reverse the level only when you’re on an even level.

### 🧠Intuition/Approach

- **If root is null, return empty result.**
- **Do BFS using a queue, level by level.**
- **Store node values of the current level in a vector.**
- **Reverse that vector if the level number is even.**
- **Append that level’s values to the final result.**
- **Increment level and repeat until queue is empty.**

### ⏱ **Time Complexity (TC)**

👉 **O(N)**
- Each node is visited once in BFS.
- Reversing a level takes **O(k)** and the sum of all k’s is **N**.
So final complexity stays **O(N)**.

### 💾 **Space Complexity (SC)**
👉 **O(N)**  
Because:
- The queue can hold up to an entire level (~N/2 nodes).
- The `v` vector stores one level at a time.
- `res` stores all nodes.
Worst-case = **O(N)**.

---

### ✅Solution
```cpp
 vector<vector<int>> zigzagLevelOrder(TreeNode* root) {
        vector<vector<int>> res;
        if(root == nullptr) return res;
        
        int level = 1;
        queue<TreeNode*> q;
        q.push(root);
        
        while(!q.empty()){
            vector<int> v;
            int size = q.size();
            for(int i=0; i<size; i++){
                TreeNode* node = q.front(); q.pop();
                v.push_back(node->val);
                if(node->left) q.push(node->left);
                if(node->right) q.push(node->right);
            }
            if(level % 2 == 0) 
                reverse(v.begin(), v.end());
                
            res.push_back(v);
            
            level++;
            
        }
        return res;
    }
```

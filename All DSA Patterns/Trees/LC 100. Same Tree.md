#### 🗝️Key Concept

- With any of the traversals, keep traversing through same nodes of the trees, at any point, if there's a difference in node/ node's value, return false and stop recursion.
- “To check if two binary trees are identical, I compare them node-by-node using recursion. If both nodes are null, that subtree is identical. If only one is null or their values differ, they aren’t identical. Otherwise I recursively check the left subtrees and right subtrees and return true only if both match. This runs in O(N) time because every node is visited once, and O(H) space due to recursion depth.”

### 🧠Intuition/Approach

- **If both nodes are null → return true.**
- **If one is null → return false.**
- **If values differ → return false.**
- **Recursively check left subtrees AND right subtrees.**

# ⏱ **Time Complexity (TC)**

👉 **O(N)**  
You compare each node once.

# 💾 **Space Complexity (SC)**

👉 **O(H)** recursion depth
- Balanced → O(log N)
- Skewed → O(N)

---

### ✅Solution

```cpp
bool isIdentical(Node* r1, Node* r2) {
        // code here
        if(r1 == nullptr && r2 == nullptr) return true;
        if(r1 == nullptr || r2 == nullptr) return false;
        
        if(r1->data != r2->data) return false;
        
        bool left = isIdentical(r1->left, r2->left);
        bool right = isIdentical(r1->right, r2->right);
        
        return (left && right);
    }
```

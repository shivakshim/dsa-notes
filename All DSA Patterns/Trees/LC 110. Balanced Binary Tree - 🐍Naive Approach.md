#### 🗝️Key Concept   

- We use double recursion :  1. the maxheight function,
					    2. The isbalanced function

- The isbalanced fun first checks if the root is balanced by finding its maxleft & maxright. Then if it is, calls isBalanced for its left & right subtrees.
- Base case of isBalanced()  ➡️ A **null node is literally an empty tree**.

### 🧠Intuition/Approach

- If node is null → **balanced → return true**
- Calculate left height
- Calculate right height
- If diff > 1 → **return false**
- Recursively check left subtree
- Recursively check right subtree
- If both are balanced → **return true**, else false

### 📦SC
- Brute - O(1)
- Optimized -  Worst-case: O(N) -> (because of recursion stack in a skewed tree)
	Best/Average: O(log N) -> (balanced tree recursion depth)

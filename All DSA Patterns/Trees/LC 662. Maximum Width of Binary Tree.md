🔗[Problem link](https://leetcode.com/problems/maximum-width-of-binary-tree/description/)

---
#### 🗝️Key Concept

- We need to give horizontal positioning to he nodes of every level, level - by - level.
- For every node, the horizontal positions of the left and right child is : 2* i & (2* i)+1.
- By this way, we can index all the nodes of all levels by BFS, and then find the 
	🌟maxwidth = max(maxwidth, rightmost-leftmost+1)

### 🧠Intuition/Approach

- Initialize queue with `(root, 0)` and `maxWidth = 0`.
- While queue is not empty
    - Record `levelSize = queue.size()`.
    - Save `leftmost = index of first node` in this level.
    - For each node in this level:
        - Pop `(node, index)` from queue.
        - Push left child `(node->left, 2*index)` if exists.
        - Push right child `(node->right, 2*index+1)` if exists.
        - Save `rightmost = index of last node` in this level.
    - Update `maxWidth = max(maxWidth, rightmost - leftmost + 1)`.
- Return `maxWidth`.

### ❓Note

- The ideal left extreme node for any level is equal to the level number, we substract this from each pos to avoid overflow and start indexing from 0 for each level.

### ⏱ **Time Complexity (TC)**

- **O(n)** — Each node is visited exactly once
### 💾 **Space Complexity**

- **O(n)** — BFS queue can store up to `n` nodes in the last level.

---

### ✅Solution

```cpp
int widthOfBinaryTree(TreeNode* root) {
        int maxwidth = 0;
        if(root==nullptr) return maxwidth;

        queue<pair<TreeNode* , int>> q;
        q.push({root, 1});

        int idealleft = 1, level=1;
        while(!q.empty()){
            int size = q.size();
            int idealleft = level;
            long long leftmost=0, rightmost = 0;
            for(int i=0; i<size; i++){
                TreeNode* node = q.front().first;
                int horlevel = q.front().second;
                q.pop();

                long long pos = horlevel-idealleft; 
                if(node->left) q.push({node->left, 2*pos});
                if(node->right) q.push({node->right, ((2*pos) + 1)});

                if(i == 0) leftmost = pos;
                if(i == size-1) rightmost = pos;

                level++;
            }
            maxwidth = max(maxwidth, (int)(rightmost-leftmost+1));
        }
        return maxwidth;
    }
```

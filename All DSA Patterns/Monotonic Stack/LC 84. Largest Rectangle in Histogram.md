🔗[Problem link](https://leetcode.com/problems/largest-rectangle-in-histogram/description/)
Given an array of integers `heights` representing the histogram's bar height where the width of each bar is `1`, return _the area of the largest rectangle in the histogram_.

---
#### ⌛Brute Force 

- Nested loops to find out next smaller to left & right.

#### 🗝️Key Concept

- **Restate the goal in simple words**
> “We want the biggest rectangle that can sit inside this histogram.”  
> Rectangle = some **height** × some **width**.

- So basically we need to check for each height, how far wide i can stretch it -> till i don't get something smaller on the left  on the right, in that case, the area of histogram of height[i] will be in **between** next smaller left & next smaller right(not included).

### 🧠Intuition/Approach

- For each bar, compute nearest smaller to left (`left[i]`) using monotonic stack.
- For each bar, compute nearest smaller to right (`right[i]`) using monotonic stack.
- If no smaller exists on a side, use sentinel: `-1` for left, `n` for right.
- For each bar `i`, width = `right[i] - left[i] - 1`.
- Area = `heights[i] * width`.
- Take the maximum area across all bars.
- “Every bar gets a chance to be the shortest wall of a rectangle; find how far it can expand using nearest smaller elements.”

### ❓Note

- the area is between the smaller left and the smaller right, so basically we say- for this height how much wider can i go till this height s the smallest -> then for that width area is that heights * width.

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(n)

---

### ✅Solution

```cpp
int largestRectangleArea(vector<int>& heights) {
        int n = heights.size();
        vector<int> left(n, -1);
        vector<int> right(n, n);

        //fill next smallest right
        stack<int> st;
        for(int i=0; i<n; i++){
            while(!st.empty() && heights[i] < heights[st.top()]){
                int idx = st.top();  st.pop();
                right[idx] = i;
            }
            st.push(i);
        }

        while(!st.empty()) st.pop();

        //fill next smallest left 
        for(int i = n-1; i>=0; i--){
            while(!st.empty() && heights[i] < heights[st.top()]){
                int idx = st.top(); st.pop();
                left[idx] = i;
            }
            st.push(i);
        }

        //calculate max area
        int maxarea = INT_MIN;
        for(int i=0; i<n; i++){
            maxarea = max(maxarea, heights[i] * (right[i] - left[i] - 1));
        }
        return maxarea;
    }
```

#### ⌛Brute Force 

- 

#### 🗝️Key Concept

- Fix a **bottom row** (call it `r`).
- For each column `c`, compute `heights[c]` = # consecutive 1s ending at row `r`.
	Now, imagine a rectangle ending at row `r`
- Its top is **implicitly defined** by the shortest column in the rectangle’s span.
- That’s because the rectangle must have 1s in every column for every row in its height.
**Key idea:**
> You don’t have to pick `top_row` explicitly because `heights[c]` already tells you “how far up can I go in this column?”

### 🧠Intuition/Approach

- Initialize compressed array = compressed(m, 0)
- For each row `r` in matrix:
    - Update `heights[c]`: `heights[c] += 1` if `matrix[r][c]=='1'`, else `heights[c]=0`.
- Call `maxarea(heights)` (LC 84 logic) → gives largest rectangle ending at row `r`.
- Track global max area across all rows.
- Return global max area.

### ❓Note

- 

### ⌛TC 
- Brute - O(n4)
- Optimized - O(n2)

### 📦SC
- Brute - O(1)
- Optimized - O(2n)

---

### ✅Solution

```cpp
 int maxarea(vector<int> &compressed){
        int n = compressed.size();
        vector<int> left(n, -1);
        vector<int> right(n, n);

        stack<int> st;
        for(int i=0; i<n; i++){
            while(!st.empty() && compressed[i] < compressed[st.top()]){
                int idx = st.top(); st.pop();
                right[idx] = i;
            }
            st.push(i);
        }

        while(!st.empty()) st.pop();

        for(int i=n-1; i>=0; i--){
            while(!st.empty() && compressed[i] < compressed[st.top()]){
                int idx = st.top(); st.pop();
                left[idx] = i;
            }
            st.push(i);
        }

        int maxarea = 0;
        for(int i=0; i < n; i++){
            maxarea = max(maxarea, compressed[i] * (right[i] - left[i] - 1));
        }
        return maxarea;
    }
    int maximalRectangle(vector<vector<char>>& matrix) {
        int n = matrix.size();
        int m = matrix[0].size();
        int ans = 0;

            vector<int> compressed(m,0);
            for(int bottom = 0; bottom < n; bottom++){
                for(int col = 0; col < m; col++){
                    if(matrix[bottom][col] == '1')
                    compressed[col] += 1;
                    else compressed[col] = 0;
                }
                ans = max(ans, maxarea(compressed));
            }
        return ans;
    }
```

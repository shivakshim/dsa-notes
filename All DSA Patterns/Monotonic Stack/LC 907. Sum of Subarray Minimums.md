🔗[Problem link](https://leetcode.com/problems/sum-of-subarray-minimums/description/)
Given an array of integers arr, find the sum of `min(b)`, where `b` ranges over every (contiguous) subarray of `arr`. Since the answer may be large, return the answer **modulo** `109 + 7`.

---

#### 🗝️Key Concept

- ##### 🧩 1. Every element `arr[i]` acts as the _minimum_ in certain subarrays.
How many subarrays?  
We count how far it can extend **to the left and to the right** until we hit a smaller (or equal) element.
##### 💬 2. Definitions
- `left[i]` → index of **previous strictly smaller** element
- `right[i]` → index of **next smaller or equal** element  
    (This asymmetry prevents double counting duplicates.
So:
- Distance to left boundary = `i - left[i]`
- Distance to right boundary = `right[i] - i`

Total subarrays where `arr[i]` is minimum =  
👉 `(i - left[i]) * (right[i] - i)`

#### 🧮 3. Formula
sum=i=0∑n−1​arr[i]∗(i−left[i])∗(right[i]−i)


### 🧠Intuition/Approach

1️⃣ Use **monotonic increasing stack** to find span of influence for each element.  
2️⃣ `left[i]` = index of **previous strictly smaller** element → pop while `>= arr[i]`.  
3️⃣ `right[i]` = index of **next smaller-or-equal** element → pop while `< arr[i]`.  
4️⃣ If none found → `left[i] = -1`, `right[i] = n`.  
5️⃣ Subarrays count where `arr[i]` is min = `(i - left[i]) * (right[i] - i)`.  
6️⃣ Contribution = `arr[i] * (i - left[i]) * (right[i] - i)`.  
7️⃣ Take mod `1e9+7`.  
8️⃣ Asymmetry (`<` vs `<=`) prevents double-counting duplicates.

### ❓Note

-  We’re not “finding mins of subarrays.”  
	We’re finding “how many subarrays treat each element as their min.”
- **Why strictly smaller on left and smaller-or-equal on right?**  
    👉 To make sure duplicates don’t double count overlapping subarrays.  
    Each duplicate “owns” its zone of influence cleanly.

### ⌛TC 
- Brute - O(n2)
- Optimized - o(n)

### 📦SC
- Brute - O(1)
- Optimized - O(n)

---

### ✅Solution

```cpp
 int sumSubarrayMins(vector<int>& arr) {
        int n = arr.size();
        vector<int> left(arr.size(), -1);
        vector<int> right(arr.size(), arr.size());
        long long ans = 0 ;
        int mod = 1e9 + 7;

        stack<int> st;
        for(int i=0; i<arr.size(); i++){
            while(!st.empty() && arr[i] <= arr[st.top()]){
                int idx = st.top(); st.pop();
                right[idx] = i;
            }
            st.push(i);
        }

        while(!st.empty()) st.pop();

          for(int i=arr.size()-1; i>=0; i--){
            while(!st.empty() && arr[i] < arr[st.top()]){
                int idx = st.top(); st.pop();
                left[idx] = i;
            }
            st.push(i);
        }

        for (int i = 0; i < n; i++) {
        long long contrib = 1LL * arr[i] * (i - left[i]) % mod * (right[i] - i) % mod;
        ans = (ans + contrib) % mod;
    }
    return ans;
    }
```

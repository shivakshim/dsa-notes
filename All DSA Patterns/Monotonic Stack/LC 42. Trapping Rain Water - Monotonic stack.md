
🔗[Problem link](https://leetcode.com/problems/trapping-rain-water/)
Given `n` non-negative integers representing an elevation map where the width of each bar is `1`, compute how much water it can trap after raining.
![[Pasted image 20251016105826.png]]

---
#### ⌛Brute Force 

- 2 arrays for maxleft & maxright (trach using 2 vars) -> 2 pointers -> monotonic stack

#### 🗝️Key Concept

- Instead of finding the water trapped per block, we find the water trapped between intervals whose next greater right we find out.
- the next greater left is simply the element below the top(curr).
- So then for that top, the water on it is (min(arr[st.top()], arr[i]) - curr * (i - st.top() - 1).
- This way we are finding out the water trapped between intervals, we don't need the absolute/global max but we just need the next greater left & right, and the algo does this- we pop when we get a greater element(maxright), and the element below this top, is the maxleft, since the stack in monotonic .

### 🧠Intuition/Approach

- **Iterate** through each bar `i`.
- **While current height > top of stack:**
    - Pop the top `curr` (the valley).
    - If stack not empty → calculate water trapped above `curr` as:  
        `(min(height[i], height[new_top]) - curr) * (i - new_top - 1)`
    - Add this to total `water`.
- **Push** current index `i` onto the stack.
- **Return** total `water` after the loop.

### ⌛TC 
- **Brute force:** `O(n^2)` (check every pair of bars for trapped water).
- **Monotonic stack:** `O(n)` (each bar pushed/popped at most once).

### 📦SC
- **Brute force:** `O(1)` or `O(n)` if prefix max arrays used.
- **Monotonic stack:** `O(n)` for the stack.

---

### ✅Solution

```cpp
int trap(vector<int>& height) {
        stack<int> st;
        int water = 0;

        for(int i=0; i<height.size(); i++){
            while(!st.empty() && height[i] > height[st.top()]){
                int curr = height[st.top()];  st.pop();
                if(!st.empty())
                water += (min(height[i], height[st.top()]) - curr) *
                                     (i - st.top() - 1);
            }
            st.push(i);
        }

        return water;
    }
```

#### 🗝️Key Concept

- You’re given multiple employees, each with their own **sorted list of busy intervals**.
- You need to find **free time intervals** when _everyone_ is free — i.e., **gaps between merged busy times** across all employees.
- The trick: merge all busy intervals into one timeline first, then find the holes.

### 🧠Intuition/Approach

- **Flatten** all employee intervals into one list.  
    → basically: combine all sublists into a single big vector.
- **Sort** that list by start time.  
    → ensures we can merge overlapping busy slots.
- **Merge overlapping intervals** (like Meeting Rooms II).  
    → maintain `prev_end`, and if next.start > prev_end → that’s a free slot!
- **Record free time gaps** between merged intervals.  
    → free = `[prev_end, curr.start]`.
- **Return** those free intervals as the answer.

### ❓Note

- For sorting and merging into 1 by start, we could have also used the PQ minheap technique to optimize the time further, like we did for merge linked lists.

### ⌛TC 
| Step       | Operation                  | Time                      |
| ---------- | -------------------------- | ------------------------- |
| Flattening | Combine all employee lists | **O(n * k)**              |
| Sorting    | Sort by start time         | **O(n * k * log(n * k))** |
| Merging    | Single pass merge          | **O(n * k)**              |
✅ **Total Time Complexity = O(n * k * log(n * k))**  
(dominated by the sorting step)
### 📦SC
| Usage                     | Space                                 |
| ------------------------- | ------------------------------------- |
| Flattened list            | **O(n * k)**                          |
| Merged list / output list | **O(n * k)** worst case (no overlaps) |
✅ **Total Space Complexity = O(n * k)**

### ✅Solution

```cpp
vector<int> findFreeIntervals(vector<vector<int>> &schedules) {
    vector<pair<int,int>> intervals;
    for(int i=0; i<schedules.size(); i++){
        vector<int> v = schedules[i];
        for(int i=0; i<v.size(); i+=2){
            int start = v[i];
            int end = v[i+1];
            intervals.push_back({start, end});
        }
    }

    //step2 : sort on start
    sort(intervals.begin(), intervals.end());
    //step3 : merge
    vector<pair<int, int>> merged;
    int start = intervals[0].first;
    int end = intervals[0].second;
    for(int i=0; i< intervals.size(); i++){
        if(intervals[i].first <= end) end = max(end, intervals[i].second);
        else
        {
            merged.push_back({start, end});
            start = intervals[i].first;
            end = intervals[i].second;
        }
    }
    merged.push_back({start, end});

    //step4 : find non overlaps
    vector<int> res;
    if(merged[0].first > 0) 
    {
        res.push_back(0);
        res.push_back(merged[0].first - 1);
    }
    end = merged[0].second;
    for(int i = 1; i<merged.size(); i++){
        if(merged[i].first - 1 >= end + 1) {
        res.push_back(end + 1);
        res.push_back(merged[i].first - 1);
        }
        end = merged[i].second;
    }
    if(end < 100000000){
    res.push_back(end+1);
    res.push_back(100000000);
    }
    return res;
}
```

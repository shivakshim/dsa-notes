🔗[Problem link](https://leetcode.com/problems/fruit-into-baskets/description/)
Given the integer array `fruits` representing fruits of different kinds(numbers), return _the **maximum** number of fruits you can pick_ if you can pick exactly 2 kinds of fruits.

---
#### 🗝️Key Concept

- Question can be reworded as - find the longest possible subarray with at most k(2) unique numbers(duplicates allowed), which is nothing but a special case of [[LC 340. Longest Substring with K Uniques]] where k=2.

---

### ✅Solution

```cpp
int totalFruit(vector<int>& fruits) {
        unordered_map<int, int> freq;
        int l=0, r=0, maxlen = 0;
        while(r < fruits.size()){
        
            freq[fruits[r]]++;

            if(freq.size() > 2){
                freq[fruits[l]]--;
                if(freq[fruits[l]] == 0) freq.erase(fruits[l]);
                l++;
            }

            if(freq.size() <= 2)   //even less tha k(2) is allowed, eg where total fruits < 2.
            maxlen = max(maxlen, r-l+1);

            r++;
        }
        return maxlen;
    }
```

🔗[Problem link](https://leetcode.com/problems/max-consecutive-ones-iii/description/)
Given a binary array `nums` and an integer `k`, return _the maximum number of consecutive_ `1`_'s in the array if you can flip at most_ `k` `0`'s.

---
#### ⌛Brute Force 

- Nested loops.

#### 🗝️Key Concept

- The question can be reframed to - Find Longest subarray with max k 0's.
- So we just have to check based on the freq of 0.
- Since its about frequency and not delicacy, we can move l lazily.

### 🧠Intuition/Approach

1. Keep 2 pointers `l, r` and a counter for zeros.  
2. Expand `r`, include `nums[r]`.  
3. If `nums[r] == 0`, increment zero counter.  
4. If zero counter > k → shrink window from `l` until zeros ≤ k.  
5. Update max length = `r - l + 1` each step.  


### ⌛TC 
- Brute - O(n2)
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(1)

---

### ✅Solution

```cpp
  int longestOnes(vector<int>& nums, int k) {
        unordered_map<int, int> freq;
        int l=0, r=0, maxlen = 0;

        while(r < nums.size()){
            freq[nums[r]]++;

            if(freq[0] > k){
                freq[nums[l]]--;
                l++;
            }

            if(freq[0] <= k)
            maxlen = max(maxlen, r-l+1);

            r++;
        }
        return maxlen;
    }
```

#### 🗝️Key Concept

- [[🪟 Sliding Window + Freq Maps Cheat Sheet]]
- fixed window of size `p.size()` over `s` + frequency array.
- Since we need to check for all chars in p, we actually need to check all fixed windows of size `p.size()` over `s` + frequency array. Compare with target freq to find anagrams.

### 🧠Intuition/Approach

###  Steps

1. **Setup freq arrays**
    - `target[26]` → frequency of chars in `p`.
    - `window[26]` → frequency of first `len(p)` chars in `s`.
2. **Check first window**
    - If `window == target`, record index `0`.
3. **Slide the window** (from index `len(p)` to `len(s)-1`):
    - Add new char: `window[s[i] - 'a']++`.
    - Remove old char: `window[s[i - len(p)] - 'a']--`.
    - Compare `window` with `target`.
    - If equal → record starting index `(i - len(p) + 1)`.
4. **Return all indices.**

### ❓Note

 - if the size of string to be checked for is smaller than the p return {}
- Arrays (size 26) beat hashmaps here.
- Comparison is O(26), which is constant → total O(n).

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(n)

### ✅Solution

```cpp
 bool checkInclusion(string s1, string s2) {
        vector<int> perm(26,0);
        vector<int> freq(26,0);

        int k = s1.length();
        if(s2.length() < k) return false;

        for(int i=0; i<k; i++){
            perm[s1[i] - 'a']++;
            freq[s2[i] - 'a']++;
        }
        if(perm == freq) return true;

        for(int i=k; i<s2.length(); i++){
            freq[s2[i-k] - 'a']--;
            freq[s2[i] - 'a']++;
            if(perm == freq) return true;
        }
        return false;
    }
```

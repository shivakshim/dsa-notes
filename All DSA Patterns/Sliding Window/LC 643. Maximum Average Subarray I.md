🔗[Problem link](https://leetcode.com/problems/maximum-average-subarray-i/description/)
Find a contiguous subarray whose **length is equal to** `k` that has the maximum average value and return _this value_.

---
#### ⌛Brute Force 

- Nested loops.

#### 🗝️Key Concept

- Just compare sums, no shrinking/growing logic.
- so we just first consider the first window of size k, then we keep removing left and adding right after that and check for maxsum.
- data type of ans and sum is double, max function works well with double.

### 🧠Intuition/Approach

- Compute sum of first `k` elements.
- Slide window by subtracting `nums[i]`, adding `nums[i+k]`.
- Track max sum → return `max_sum / k`.

### ❓Note

- Fixed-size sliding window → works fine even with negatives, since all windows are exactly size `k`.

### ⌛TC 
- Brute - O(n2)
- Optimized - (n)

### 📦SC
- Brute - O(1)
- Optimized - O(1)

---

### ✅Solution

```cpp
 double findMaxAverage(vector<int>& nums, int k) {
        double sum = 0;

        for(int i=0; i<k; i++) sum += nums[i];
        double maxsum = sum/k;

        for(int i = k; i < nums.size(); i++){
            sum += nums[i] - nums[i-k];
            maxsum = max(maxsum, sum/k);
        }

        return maxsum;
    }
```

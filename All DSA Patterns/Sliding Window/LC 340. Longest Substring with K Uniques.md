 Your task is to find the **length** of the **longest substring** that contains exactly **k** distinct characters.

---
#### ⌛Brute Force 

- Nested loops.

#### 🗝️Key Concept

- **Rule of thumb:
- If the problem is about **duplicates / last seen position** → _Index Map_.
- If the problem is about **distinct counts / k limit** → _Frequency Map_.
- Here since we don't care about duplicates, we don't care about their indices but the overall distinct char, so we take a frequency map.

### 🧠Intuition/Approach

- Initialize `l = 0, r = 0, maxLen = 0`, and a frequency map.
- Expand window by adding `s[r]` into the map.
- If map size > k → shrink from left: decrement `s[l]`, erase if count hits 0, then move `l++`
- Update `maxLen = max(maxLen, r - l + 1)` whenever map size ≤ k.
- Continue until `r` reaches end of string.
- Return `maxLen`.

### ❓Note

- We move l lazily because we need the max length, since we already got a l-r window, even if we do shrink l we will always get length less.

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(1)

---

### ✅Solution

```cpp
int longestKSubstr(string &s, int k) {
        // code here
        int l = 0, r = 0, maxlen = -1;
        unordered_map<int, int> mp;
        
        while(r < s.length()){ 
              mp[s[r]]++;
        
              if(mp.size() > k)
              {
                mp[s[l]]--;
                if(mp[s[l]] == 0) mp.erase(s[l]);
                l++;
              }
        
              if(mp.size() == k) 
              maxlen = max(maxlen, r-l+1);
            
            r++;
        }
        return maxlen;
    }
```

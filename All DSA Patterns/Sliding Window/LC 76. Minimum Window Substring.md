🔗[Problem link](https://leetcode.com/problems/minimum-window-substring/description/)
Find the smallest substring of `s` that contains all characters of `t` (with correct frequencies).

---

#### 🗝️Key Concept

- ⚡Trick : 2 hashmaps + count variable.
- Hashmaps -> To track the duplicates , `cnt`  -> To track if all elements of t are present in window. Why?
	👉Because just by the freq maps, we cannot tell if all elements are covered, freq can only tell if the number of duplicates per char are covered.

### 🧠Intuition/Approach

1. Keep adding s[r] in the window freq map, window[s[r]]++.
2. if this s[r] is a char in t, check if its freq is <= freq that is needed, if yes cnt++.
3. if window becomes valid i.e. cnt = t.size() , record it, start shrinking from left.
4. Every time we shrink, do window[s[l]]--, if this is preset in needed(t), and its freq has become less than required then do cnt--, window has become invalid.
5. When recording the window, store the start & minlen.
6. Once done, return s.str(start, minlen).

### ❓Note

- 2 maps are crucial here: one “recipe” (`need`), one “current pantry” (`window`).
- This is a **variable-sized sliding window with shrink**.

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(2n)

---

### ✅Solution

```cpp
 string minWindow(string s, string t) {
        unordered_map<char, int> needed, window;
        for(auto x: t) needed[x]++;

        int l = 0, r = 0, cnt = 0, start = 0, minlen = INT_MAX;
        while(r < s.length()){

            window[s[r]]++;

            if(needed.count(s[r]) && window[s[r]] <= needed[s[r]]){
                cnt++;
                }

            while(cnt == t.size()){
                if(minlen > r-l+1) 
                {
                    minlen = r-l+1;
                    start = l;
                }
                window[s[l]]--;
                if(needed.count(s[r]) && window[s[l]] < needed[s[l]]) cnt--;
                l++;
            }
            r++;
        }
       return minlen == INT_MAX ? "" : s.substr(start, minlen);
    }
```

#### 🗝️Key Concept

- [[🪟 Sliding Window + Freq Maps Cheat Sheet]]
- fixed window of size `p.size()` over `s` + frequency array.
- Since we need to check for all chars in p, we actually need to check all fixed windows of size `p.size()` over `s` + frequency array. Compare with target freq to find anagrams.

### 🧠Intuition/Approach

- **Setup freq arrays**
    - `target[26]` → frequency of `s1`.
    - `window[26]` → frequency of first `len(s1)` chars in `s2`.
- **Check first window**
    - If `window == target`, return `true`.
- **Slide the window** (from index `len(s1)` to `len(s2)-1`):
    - Add new char: `window[s2[i] - 'a']++`.
    - Remove old char: `window[s2[i - len(s1)] - 'a']--`.
    - Compare `window` with `target`.
    - If equal → return `true`.
- **If loop finishes with no match → return false.**

### ❓Note

- if the size of string to be checked for is smaller than the anagram return {}

### ✅Solution

```cpp
 vector<int> findAnagrams(string s, string p) {
        vector<int> anagram(26,0);
        vector<int> freq(26,0);

        int k = p.size();
        if(s.length() < k) return {};

        for(int i=0; i<k; i++){
            anagram[p[i] - 'a']++;
            freq[s[i] - 'a']++;
        }

        vector<int> ans;
        if(anagram == freq) ans.push_back(0);
        for(int i=k; i<s.length(); i++){
            freq[s[i-k] - 'a']--;
            freq[s[i] - 'a']++;
  
            if(anagram == freq) ans.push_back(i-k+1);
        }
        return ans;
    }
```

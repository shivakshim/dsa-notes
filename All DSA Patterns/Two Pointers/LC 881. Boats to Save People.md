🔗[Problem link](https://leetcode.com/problems/boats-to-save-people/description/)
Return the minimum number of boats to carry every given person, with weight denoted by people[i], where each boat can have at max 2 people and limit of weight.

---

### 🧠Intuition

- Each boat: **max 2 people + weight limit**.
- To ensure minimum boats we pair lightest with the heaviest.
- So we will sort the weight array, keep converging pointers. At any point we get people[i]+people[j] as the max weight possible. 
- So if this weight is <= limit, we move for next pairs.
- j--, boat++; will always happen because even if the less, this was the max possible. And even for less sum we will have to get a boat since boat can only have 2 people, even if the sum < limit, or if its more, then the heaviest will get the boat.

### ⌛TC 
- Brute - 
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(1)

---

### ✅Solution

```cpp
int numRescueBoats(vector<int>& people, int limit) {
        sort(people.begin(), people.end());
        int boats = 0;
        int i = 0; int j = people.size()-1;
        while(i <= j){
          int weight = people[i] + people[j];
          if(weight <= limit){
            i++;
          }
          j--; //if weight>limit, we send the heaviest alone, since it had lighest posible
          boats++;
        }
      return boats;
    }
```

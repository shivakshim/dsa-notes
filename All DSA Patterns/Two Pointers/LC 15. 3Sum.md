🔗[Problem link](https://leetcode.com/problems/3sum/description/)
Given an integer array nums, return all the triplets ([nums[i], nums[j], nums[k]]) such that i!=j!=k.

---

### 🧠Intuition

- if we fix nums[i] -> this reduces to 2sum.
- So we sort the array to perform 2 pointer approach on 2sum. on the remaining range.
- 1st for loop -> the fixed element. 2nd while loop -> remaining range for 2sum.
- We just need to check the remaining range because all previous triplets would already be calculated.

#### ❓Note 

- Since we need at least 2 elements in range we only go from i=0 -> n-3.
- We must take care of duplicates at moving i and when we get a particular j, k. Move j and k till its duplicate.
     -- if(i>0 && nums[i] == nums[i-1]) continue;
     --  ans.push_back({nums[i], nums[j], nums[k]});
                j++; k--;
                while(j<k && nums[j] == nums[j-1]) j++;
                while(j<k && nums[k] == nums[k+1]) k--;


### ✅Solution

```cpp
 vector<vector<int>> threeSum(vector<int>& nums) {
        int n = nums.size();
        vector<vector<int>> ans;
        sort(nums.begin(), nums.end());
        
        for(int i=0; i < n-2; i++){
            if(i>0 && nums[i] == nums[i-1]) continue;   //if this is same as last
            int j = i+1;  int k = n-1;
            while(j < k){
                int sum = nums[i] + nums[j] + nums[k];
                if(sum == 0){
                ans.push_back({nums[i], nums[j], nums[k]});
                j++; k--;
                
                while(j<k && nums[j] == nums[j-1]) j++;  //move till not duplicate
                while(j<k && nums[k] == nums[k+1]) k--;  //move till not duplicate
                }
                
                else if(sum < 0) j++;
                else k--;
            }
        }
        return ans;
    }
```

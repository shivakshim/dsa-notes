🔗[Problem link](https://leetcode.com/problems/sort-array-by-parity/description/)
Given an integer array `nums`, move all the even integers at the beginning of the array followed by all the odd integers.

---
#### ⌛Brute Force 

- Take another ans array, loop twice , once for even once for odd.

#### 🗝️Key Concept

- Use Staggered pointers - 1 writes(slower), 1 scans(faster).
- i on odd, j moves to even, swap. Repeat.

### 🧠Intuition

- We will keep i at the 1st odd, j ahead of i.
- move j till it reaches even , swap the two, move both by 1.
- This makes sure i is always on the odd number to be replaced, and we move j to reach even.
- Repeats till j reaches n.

### ⌛TC 
- Brute - O(n)
- Optimized - O(n)

### 📦SC
- Brute - O(n)
- Optimized - O(1)

---

### ✅Solution

```cpp
vector<int> sortArrayByParity(vector<int>& nums) {
        int i = 0;
        while(i< nums.size() && nums[i] % 2 == 0){
           i++;
        }
        int j = i+1;
        while(j < nums.size()){
            if(nums[j] % 2 == 0){ //j is even
                swap(nums[i], nums[j]);
                i++;
            }
            j++;
        }
        return nums;
    }
};
```

🔗[Problem link](https://leetcode.com/problems/merge-sorted-array/description/)
Merge 2 sorted arrays in place, without extra space.

---
#### ⌛Brute Force - 

  1. append nums2 to nums1. then sort function.
  2. for every smallest element, shift next ones by 1.

### 🧠Intuition

- Instead of Merging from front, we will merge from the end, Why?  
  because merging from front becomes messy since we need to merge it in place, and we wouldn't certainly know what to move and where. 
- BUT, if we start merging from back, comparing last, second last and so on elements of arrays, we know that we are getting the correct certain largest elements.
- For implementing this, we keep 3 pointers -
1. i = m-1 (end of array 1)
2. j = n-1 (end of array 2)
3. k = m+n-1 (end of merged array where we need to place the largest)  
-  For every iteration -> We check which end is larger -> place it in k -> decrement index of larger and k. Goes on till any one array isn't over.  
- If i<0 we simply then run a loop to copy remaining elements of j to k.  
  else do nothing (since elements of nums1 are already present).

### ❓Note

- The 2nd part where if i<0 we paste all elements of j handles the edge cases too.

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n+m)

### 📦SC
- Brute - O(n+m)
- Optimized - O(1)

---

### ✅Solution

```cpp
void merge(vector<int>& nums1, int m, vector<int>& nums2, int n) {
        int i = m-1, j = n-1, k = m+n-1;
        while(i >= 0 && j >= 0){
            if(nums1[i] >= nums2[j]){
                 nums1[k] = nums1[i];  //k holds the bigger element
                 k--; i--;                //decrement bigger
            }
            else{
                nums1[k] = nums2[j];
                k--; j--;
            }
        }
        if(i < 0){
            while(j >= 0){
                nums1[k] = nums2[j];
                k--; j--;
            }
        }
    }
```

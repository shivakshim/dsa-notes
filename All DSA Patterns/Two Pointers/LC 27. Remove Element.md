🔗[Problem link](https://leetcode.com/problems/remove-element/description/)
Given an integer array `nums` and an integer `val`, remove all occurrences of `val` in `nums` [**in-place**](https://en.wikipedia.org/wiki/In-place_algorithm).

---
#### ⌛Brute Force 

- O(n2)

### 🧠Intuition

- We use Staggered pointers, j scans , i writes.
- We keep a variable k for indices of our array, arr[k] will store the elements we want i.e., other than val.
- For every element:
    - If it’s **not** `val` → copy it to position `k` → increment `k`.
    - If it **is** `val` → skip it.
- End result: first `k` elements are your answer, everything after is irrelevant.

### ❓Note

- since we have a k++, in the end of the loop which will also run the last time, k will be the answer

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(1)

---

### ✅Solution

```cpp
 int removeElement(vector<int>& nums, int val) {
        int k = 0;
        for(int i=0; i< nums.size(); i++){
            if(nums[i] != val) {
            nums[k] = nums[i];
            k++;
            }
        }
        return k;
    }
```

Given an array of integers `nums` and `target`, return _indices of the two numbers such that they add up to `target`_.

---

### 🧠Intuition
- variation : 2 pointers -> Converging
- we can start from both ends and move inward based on sum comparison (left++ if sum < target, right-- if sum > target).

---

### ✅Solution

```cpp
vector<int> twoSum(vector<int>& numbers, int target) {
       int i = 0;
       int j = numbers.size() - 1;
       while(i < j){
        int sum = numbers[i] + numbers[j];
        if(sum == target) return {i+1,j+1};
        else if(sum < target) i++;
        else j--;
       }
       return {};
    }
```

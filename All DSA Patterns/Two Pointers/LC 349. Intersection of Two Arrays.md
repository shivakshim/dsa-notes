🔗[Problem link](https://leetcode.com/problems/intersection-of-two-arrays/description/)
Return vector of intersection of 2 arrays (only unique).

---
#### ⌛Brute Force 

-  **Approach: Using `unordered_set`**
1. Put all elements of the smaller array in an `unordered_set` (for O(1) average lookup).
2. Iterate over the larger array and check if each element exists in the set.
3. If it exists, add it to a result set (to avoid duplicates).
4. Convert that set to a vector and return.

### 🧠Intuition

-  We will first sort both elements.
- We keep 2 pointers on start of both arrays.
- We have to now make sure , we only check logic for the 1st unique element and not a duplicate.
- Once this is ensured, We will simply check -  
    1. If the elements are same -> push_back in answer  
    2. If nums1[i] < nums2[j] -> increment i to find potential  
    greater equal element to nums2[j].  
    3. Else j++.
- Loop continues till the smaller array ends.

### ❓Note

- In  [LC 350. Intersection of Two Arrays II](https://leetcode.com/problems/intersection-of-two-arrays-ii/)- Just remove check for duplicates.


### ⌛TC 
- Brute - O(n+m)
- Optimized - O(nlogn + mlogm)

### 📦SC
- Brute - O(min(n,m))
- Optimized - O(1)

---

### ✅Solution

```cpp
vector<int> intersection(vector<int>& nums1, vector<int>& nums2) { 
  sort(nums1.begin(), nums1.end()); 
  sort(nums2.begin(), nums2.end()); 
  vector<int> ans; 
  int i = 0, j = 0; 

  while(i < nums1.size() && j < nums2.size()){ 
	      if(i>0 && nums1[i] == nums1[i-1]) {i++; continue;} 
	      if(j>0 && nums2[j] == nums2[j-1]) {j++; continue;} 

		  if(i < nums1.size() && j < nums2.size()){ 
		  if(nums1[i] == nums2[j]) { 
			  ans.push_back(nums1[i]); 
			  i++; j++;
			} 
		  else if(nums1[i] < nums2[j]) i++; 
		  else j++;
		 } 
		} 
	return ans; 
	} 
```
 
 #### ⌛Brute force

```cpp
vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
       // to always assume that nums1 is smaller, we will swap if nums2 is smaller
       if(nums1.size() > nums2.size()) 
       swap(nums1, nums2);  //takes O(1)

       unordered_set<int> s(nums1.begin(), nums1.end());
       unordered_set<int> ans;

       for(int i = 0; i< nums2.size(); i++){
        if(s.count(nums2[i])) ans.insert(nums2[i]);
       }

       return vector<int>(ans.begin(), ans.end());
    }
```

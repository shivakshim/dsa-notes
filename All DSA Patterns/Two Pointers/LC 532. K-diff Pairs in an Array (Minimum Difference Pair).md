🔗[Problem link](https://leetcode.com/problems/k-diff-pairs-in-an-array/)
Given an array of integers `nums` and an integer `k`, return _the number of **unique** k-diff pairs in the array_. abs(diff).

---
#### ⌛Brute Force 

- Use a map to store the elements, and just like 2 sum, here we check is diff - arr[i] is present in the map.

### 🧠Intuition

- To not use extra map - 2 pointers O(1).
- We will sort the array. then keep 2 pointers i=0, j=i+1.
- Diff is taken for every i , j pair, i takes care of duplicates by continuing till i points to a new unique element of pair.
- if(diff == k) ans++ , j++, j++. else if diff < k j++, else move i++.

### ❓Note

- since sometimes i might move ahead of j in case of multiple duplicates, while must be for both i & j < n.

### ⌛TC 
- Brute - O(n)
- Optimized - O(n + nlogn)

### 📦SC
- Brute - O(n) (map)
- Optimized - O(1)

---

### ✅Solution

```cpp
 int findPairs(vector<int>& nums, int k) {
        int n = nums.size();
        int ans=0, i=0, j=1;
        sort(nums.begin(), nums.end());
        while(i< n && j < n){
           if(i > 0 && nums[i] == nums[i-1]) {i++; continue; }
           if(i == j) {j++; continue;}
           int diff = nums[j] - nums[i];
           if(diff == k) {
            ans++;
            i++; j++;
           }
           else if(diff < k)  j++;
           else i++;
        }
        return ans;
    }
```

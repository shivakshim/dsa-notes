Find two lines that together with the x-axis form a container, such that the container contains the most water.

---

### 🧠Intuition

- We want to return max area = min(height[l], height[r]) * (r-l).
- Since this involves width of array, we wouldn't sort since we want the max width. 
- So we just keep converging pointers (for max width), and move the shorter one, since longer would contribute to max area.

### ❓Note

- When asked for width, no need to sort, when asked for max sum/product/anything of actual elements->sort.

### ⌛TC 
- Brute - O(N2)
- Optimized - O(N)

### 📦SC
- Brute - O(1)
- Optimized - O(1)

---

### ✅Solution

```cpp
int maxArea(vector<int>& height) {
        int ans = 0;
        int l = 0;
        int r = height.size() - 1;

        while(l < r){
            ans = max(ans, min(height[l], height[r]) * (r-l));
            if(height[l] < height[r]) l++;
            else r--;
        }
        return ans;
    }
```

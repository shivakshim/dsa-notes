
🔗[Problem link](https://leetcode.com/problems/3sum-closest/description/)
Given an integer array nums (n) and an integer target, return sum of three integers in nums such that the sum is closest to target. Unique solution exists.

---

### 🧠Intuition

- Similar to [[LC 15. 3Sum]], instead of 0 we are interested in min abs diff = abs(target - sum).
- After each comparison, we update j/k as required.
- Fix i, check for every j, k for smallest diff.

### ❓Note

- Since there's unique solution , no need to worry about duplicates.
- i still goes from 0 to n-3.

### ⌛TC 
- Brute - O(N3)
- Optimized - O(N2) :  (O(N2) + O(NlogN))

### 📦SC
- Brute - O(1)

---

### ✅Solution

```cpp
int threeSumClosest(vector<int>& nums, int target) {
        int n =  nums.size();
        int ans = INT_MAX;
        int mindiff = INT_MAX;
        sort(nums.begin(), nums.end());
        
        for(int i=0; i< n-2; i++)
        {
            int j = i+1;
            int k = n-1;
            while(j<k){
                int sum = nums[i] + nums[j] + nums[k];
                int diff = abs(target - sum);
                if(diff < mindiff) {
                    ans = sum;
                    mindiff = diff;
                }

                if(sum < target) j++;
                else k--;
            }
        }
        return ans;
    }
```

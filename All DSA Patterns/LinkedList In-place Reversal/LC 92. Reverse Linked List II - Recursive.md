#### 🗝️Key Concept

- **Recursively move `head` to the start of the sublist (`left`-th node).**
- Once at `left`, reverse the next `right-left+1` nodes using a helper (`reverseN`).
- Reconnect the reversed sublist with the rest of the list on the way back.

### 🧠Intuition/Approach

- **Base case:** if `left == 1`, reverse first `right` nodes via `reverseN` and return new head.
- **Recursive step:** move `head` one node forward, decrement `left` and `right` by 1.
- **Reconnect:** after recursion, set `head->next` to the returned head of the reversed sublist.
- **Return:** always return the current head (or new head if at base case).

### ❓Note

- In 1 based indexing to reach an element n , we only n-- till we go till 1, not 0.

|Complexity|Iterative|Recursive|
|---|---|---|
|Time|O(n)|O(n)|
|Space|O(1)|O(n) stack frames for recursion|

---

### ✅Solution

```cpp
 ListNode* successor;
    ListNode* reverseN(ListNode* head, int n){
        if(n==1) {
            successor = head->next;
            return head;
        }
        ListNode *newHead = reverseN(head->next, n-1);
        head->next->next = head;
        head->next = successor;
        return newHead;
    }

    ListNode* reverseBetween(ListNode* head, int left, int right) {
        if(left == 1){
         return reverseN(head, right);
        }
        head->next = reverseBetween(head->next, left-1, right - 1);
        return head;
    }
```

#### 🗝️Key Concept

- Same as [[LC 92. Reverse Linked List II - ITERATIVE]] , Just need to add a condition to check if the next node has k elements.

### 🧠Intuition/Approach

- ```cpp
  while(true){
            ListNode *node = beforeleft->next;
            for(int i=1; i<k && node; i++){
                node = node->next;
            }
            if(!node) break;
  ```
- To check if the next list is available, otherwise do nothing.
- This works because after each list is reversed, we take care of attaching the beforeleft to newhead and tail to next list's start. So till the next list starts, everything is sorted till there.

### ✅Solution

```cpp
  ListNode* reverseKGroup(ListNode* head, int k) {
        if(!head || !head->next) return head;
        ListNode* dummy = new ListNode(0, head);
        ListNode* beforeleft = dummy, *p = head;

        while(true){
            ListNode *node = beforeleft->next;
            for(int i=1; i<k && node; i++){
                node = node->next;
            }
            if(!node) break;

            ListNode *p = beforeleft -> next;
            ListNode *q = nullptr, *r = nullptr;
            ListNode *tail = p;

            for(int i=1; i <= k; i++){
                r = q;
                q = p;
                p = p->next;
                //if(!p) break;
                q -> next = r; 
            }

            beforeleft -> next = q;
            tail -> next = p;
            beforeleft = tail;
        }

        return dummy -> next;
    }
```

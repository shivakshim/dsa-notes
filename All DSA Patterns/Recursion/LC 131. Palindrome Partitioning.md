#### 🗝️Key Concept

- We make partitions, ***level by level.
- First level is : just 1 partitioner which can be anywhere from after 1st element till after last.
- When we get a valid palindromic partition, we then go deeper and partition it more till we can, which means whenever i see a partition at a level being a valid palindrome, ***we try to make valid partitions in the front remaining porting of the string.
- Since we only move forward in recursion when we actually do get a valid palindromic portion, if we are able to reach th end of the string i.e i == n, we know that we have all valid partitions, and we push it in results.

### 🧠Intuition/Approach

- **Loop `i` from `idx` to end** to choose every possible substring starting at `idx`.
- **If substring `s[idx…i]` is a palindrome → push it** into the current path.
- **Recurse from `i + 1`** to partition the rest of the string.
- **Pop** the substring to undo the choice and try a different cut.

### ❓Note

**Why do we pop? (tiny note)**
- Because each recursive call represents **one possible cut**.  
	After exploring one partition path, you must **remove that substring** to try the next possible cut.  
	If you don’t pop, your “current partition path” becomes a messy bhaji with leftovers from previous branches.
	Pop = _clean slate for the next option_.

### ⌛TC 
- Optimized - TC=O(n⋅2n)

### 📦SC
- Optimized - SC=O(n)(excluding the result list)

### ✅Solution

```cpp
  bool isPalindrome(string s){
        int i = 0, j = s.length()-1;
        while(i <= j){
            if(s[i] != s[j]) return false;
            i++; j--;
        }
        return true;
    }
    void helper(int idx, int n, string& s, vector<string>& v, vector<vector<string>>& res){
        if(idx == n){
            res.push_back(v);
            return;
        }
        for(int i=idx; i<n ; i++){
            if(isPalindrome(s.substr(idx, i - idx + 1))){
                v.push_back(s.substr(idx, i - idx + 1));
                helper(i+1, n, s, v, res);
                v.pop_back();
            }
        }
    }
    vector<vector<string>> partition(string s) {
        vector<vector<string>> res;
        vector<string> v;
        int n = s.length();
        helper(0, n, s, v, res);
        return res;
    }
```

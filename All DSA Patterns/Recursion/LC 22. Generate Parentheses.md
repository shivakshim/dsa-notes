#### 🗝️Key Concept

- We need to create subsets with possible choices for each index as '(' or ')'.
- So we follow the format, using int open and close.
- Base case : open+close == 2 * n.
- Conditions to create valid parenthesis : open < n && close < open (because we shouldn't take a close if it doesn't have an open. : eg. ( ( ) ) ( - might lead to this if we don't write close < open.

### 🧠Intuition/Approach

- **Add `'('` whenever you still have opens left.**
- **Add `')'` only if closes are less than opens.**
- **When open + close = 2*n → yeet the string into the result.**
- **Backtrack after every add .**

### ❓Note

- Here the main rules for valid creation -> open < n && close < open.


- **Time Complexity:**  
    **O(Cₙ · n)** where Cₙ is the nth Catalan number ≈ **O(4ⁿ / √n)**
- **Space Complexity:**  
    **O(n)** auxiliary (due to recursion),  
    **O(Cₙ · n)** including output.

---

### ✅Solution

```cpp
  void helper(int open , int close, int n, string &s, vector<string>& res){
        if(open + close == 2*n) {
            res.push_back(s);
            return;
        }

        if(open < n){
            s += '(';
            helper(open+1, close, n, s, res);
            s.pop_back();
        }
        if(close < open){
            s += ')';
            helper(open, close+1, n, s, res);
            s.pop_back();
        }
    }

    vector<string> generateParenthesis(int n) {
        vector<string> res;
        string s = "(";
        helper(1, 0, n, s, res);
        return res;
    }
```

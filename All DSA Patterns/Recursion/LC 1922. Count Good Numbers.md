#### 🗝️Key Concept

- Instead of checking, we create all the valid strings.
- We do this by only considering the allowed digits at the required indices.

### 🧠Intuition/Approach

- Base case : is idx = n , we have created return 1, because we have reached till the end, so add 1 valid substring.
- We set res = 0, for each index, it will be returned after counting all valid combinations till that index.
- For all indices ahead of it, their results are added to this, for every possible value of this index.
- if(i is even) for {0, 2, 4, 6, 8} call res = res + helper(n, i+1)
- if(i is odd) for {2, 3, 5, 7} call the res = res + helper(n , i+1)

### ✅Solution

```cpp
    int mod = 1e9+7;
    int helper(long long n, long long i){
        if(i == n) return 1;
        int res = 0; //for every index
        if(i % 2 == 0){
            for(int digit : {0,2,4,6,8}){
                res = (res + helper(n, i+1)) % mod;   //count all unique substrings including m
            }
        }
         else if(i % 2 == 1){
            for(int digit : {2,3,5,7}){
                res = (res + helper(n, i+1)) % mod;   //count all unique substrings including m
            }
        }
        return res;
    }
    int countGoodNumbers(long long n) {
        return helper(n, 0);
    }
```

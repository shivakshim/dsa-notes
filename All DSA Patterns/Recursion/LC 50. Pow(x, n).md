#### ⌛Brute Force 

- Simple recursion.

#### 🗝️Key Concept

- **Power Function (x^n) – Key Concept**
**Key idea:**
> Use **exponentiation by squaring** to reduce recursion depth from `O(n)` → `O(log n)` and handle negative powers safely.

### 🧠Intuition/Approach

- Promote `n` to `long long` to handle `INT_MIN`.
- If `n < 0`, invert `x` → `x = 1 / x`, `n = -n`.
- Recursively compute `half = pow(x, n / 2)`.
- If `n` is even → return `half * half`.
- If `n` is odd → return `half * half * x`.

### ❓Note

- INT_MIN edge case → always use `long long` for `n`.
- `x = 0` and `n < 0` → division by zero, handle explicitly.
- Recursion depth = log2(n), safe even for 10^9.
- Works for both positive & negative exponents.

### ⌛ **Time & Space Complexity**

|Method|Time|Space (stack)|
|---|---|---|
|Recursion (fast)|O(log n)|O(log n)|
|Iterative|O(log n)|O(1)|

### ✅Solution

```cpp
  double recPow(double x, long long n){
        if(n == 0) return 1;
        if(n % 2 == 1) return x*recPow(x*x, n/2);
        else return recPow(x*x, n/2);
    } 

    double myPow(double x, int n) {
      long long N = (long long)n;
      if(N >= 0) return recPow(x, N);
      else return 1.0/recPow(x, -N);
    }
```

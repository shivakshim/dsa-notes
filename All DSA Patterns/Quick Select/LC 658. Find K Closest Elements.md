#### ⌛Brute Force 

- Min heap of size k for abs(nums[i] - x)

#### 🗝️Key Concept

- When The pivotindex/sortedindex becomes == k, we know that the left part has all smaller elements.
- Since we want the k smallest differences we can perform quickselect till i get pivotindex == k.

### 🧠Intuition/Approach

- **Transform problem** → turn each element into `(abs(x - target), value)` pairs.
- **Pick a pivot** → usually last element of current subarray.
- **Partition array** → move elements **≤ pivot** to the left, elements **> pivot** to the right.
- **Check pivot index**:
    - If `pivotIndex == k-1` → first k elements are the k closest, stop.
    - Else recurse into the side that contains the k-th element.
- **After Quickselect** → first k elements in the array are the k closest elements.
- **Sort result** → for returning in ascending order if required.

### ❓Note

- Duplicates: Use `<= pivot` in partition to handle duplicates correctly. Because since we pushed in sorted manner of elements, the diff with the smaller element would come first, now suppose out pivotindex element has a duplicate then it would push the one with smaller index to the right, and given itself in the answer with is wrong.
- Quickselect doesn’t fully sort the array — just ensures **first k elements are correct**.
- Time complexity: average **O(n)**, worst **O(n²)** (rare, depends on pivot choice).
- Space: O(1) extra if done in-place (ignoring recursion stack)

### ✅Solution

```cpp
 void quickselect(vector<pair<int,int>>& arr, int left, int right, int k){
        if(left >= right) return; // base case

        int pivot = arr[right].first;
        int sortedindex = left;

        for(int i=left; i<right; i++){
            if(arr[i].first <= pivot){
                swap(arr[i], arr[sortedindex]);
                sortedindex++;
            }
        }
        swap(arr[right], arr[sortedindex]);

        if(sortedindex == k) return;
        else if(sortedindex > k) quickselect(arr, left, sortedindex-1, k);
        else quickselect(arr, sortedindex+1, right, k);
    }

    vector<int> findClosestElements(vector<int>& arr, int k, int x) {
        vector<pair<int,int>> nums;
        for(auto a : arr){
            nums.push_back({abs(a - x), a});
        }

        quickselect(nums, 0, nums.size()-1, k-1);

        vector<int> ans;
        for(int i=0; i<k; i++){
            ans.push_back(nums[i].second);
        }

        sort(ans.begin(), ans.end()); // sort by original value
        return ans;
    }
```

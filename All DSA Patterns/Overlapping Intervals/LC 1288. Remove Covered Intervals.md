#### 🗝️Key Concept

- **Type:** Interval / sorting / greedy
- **Goal:** Remove intervals that are **completely covered by another interval**
- **Core idea:**
    - Sort intervals by **start ascending**.
        - Tie-break: **end descending** (so longer intervals come first if start is same).
    - Iterate through intervals and **keep track of the farthest end seen** (`maxEnd`).
    - If current interval’s end ≤ `maxEnd` → it’s covered → skip
    - Else → update `maxEnd` → count this interval

### 🧠Intuition/Approach

1. Sort intervals by **start ascending**, tie → **end descending**
2. Initialize `maxEnd = 0`, `count = 0`
3. For each interval `[s, e]` in sorted list:
    - If `e <= maxEnd` → interval is covered → skip
    - Else → interval is **not covered** → count++, update `maxEnd = e`
4. Return `count`

> Optional variation: return `intervals.size() - count_covered` if you count the covered intervals instead.


## **Time & Space Complexity**

- **Time Complexity:**
    - Sorting → O(nlog⁡n)O(n \log n)O(nlogn)
    - Iterating → O(n)O(n)O(n)  
        ✅ Total: O(nlog⁡n)O(n \log n)O(nlogn)
- **Space Complexity:**
    - If sorting in-place → O(1)O(1)O(1) extra
    - If copying → O(n)O(n)O(n)

### ✅Solution

```cpp
 int removeCoveredIntervals(vector<vector<int>>& intervals) {
    // Sort by end ascending, start descending if ends are equal
    sort(intervals.begin(), intervals.end(), [](const vector<int>& a, const vector<int>& b) {
        if(a[0] == b[0]) return a[1] > b[1];
        return a[0] < b[0];
    });

    int ans = intervals.size();
    int i = 0, j=1;
    while(j < intervals.size()){
        if(intervals[j][0] < intervals[i][1] && 
        intervals[i][1] >= intervals[j][1]){
            ans--;
        }
        else{
            i = j;
        }
        j++;
    }
    return ans;
    }
```

🔗[Problem link](https://leetcode.com/problems/maximum-score-of-spliced-array/description/)
Return maxsum between both arrays after swapping a range for >=1 times.

---
#### ⌛Brute Force 

- nested loops.

#### 🗝️Key Concept

💡 For both arrays- 
	Total sum ➡️ gain array(nums2 - nums1) ➡️ Kadane on gains(max gain subarray) ➡️ sum+maxgain.
	⚡ *ans = max(sum after gain of nums1 + sum after gains of nums2).
- Think of it like: “How much can I **increase nums1’s sum** by swapping some slice from nums2?”
- For each index `i`, the **gain** if you swap that element = `nums2[i] - nums1[i]`.
- So, the problem reduces to: **find a contiguous slice of this gain array with the maximum sum** → exactly **Kadane!**

### 🧠Intuition/Approach

- Compute **total sum of nums1** → this is your base.
- Compute **gain array** = `nums2[i] - nums1[i]` → what each swap contributes.
- Run **Kadane on gain array** → gives **max possible gain**.
- Maximum sum after swap = `total sum of nums1 + max_gain`.
- Repeat same for nums2, *ans = max(sum after gain of nums1 + sum after gains of nums2).

### ❓Note

- The trick is realizing **Kadane doesn’t need to know the original arrays** — it only cares about **how much each index contributes to the gain if you swap it**.

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(1)

---

### ✅Solution

```cpp
int kadane(vector<int> &gains){
        int currsum = gains[0], maxsum = gains[0];
        for(int i = 1; i < gains.size(); i++){
            currsum = max(gains[i], currsum + gains[i]);
            maxsum = max(maxsum, currsum);
        }
        return maxsum;
    }

    int maximumsSplicedArray(vector<int>& nums1, vector<int>& nums2) {
        int sum1 = 0, sum2 = 0 ,n = nums1.size();
        //step1 : find sum of both arrays
        for(int i=0; i<n ; i++){
            sum1 += nums1[i];
            sum2 += nums2[i];
        }
        //finding sum after gains for nums1
                //step2: find gains array
                vector<int> gains(n);
                for(int i=0; i<n; i++)  gains[i] = nums2[i] - nums1[i];

                //step3 : Find max subarray sum of gains(kadane)
                int sum1aftergains = sum1 + kadane(gains);

        //finding sum after gains for nums2
                for(int i=0; i<n; i++)  gains[i] = nums1[i] - nums2[i];

                //step3 : Find max subarray sum of gains(kadane)
                int sum2aftergains = sum2 + kadane(gains);

        //return max of both
        return max(sum1aftergains, sum2aftergains);
        }
```

🔗[Problem link](https://leetcode.com/problems/maximum-sum-circular-subarray/description/)
Find maxsubarray sum in a circular array.

---
#### ⌛Brute Force 

- nested loop with Kadane for each element as starting.

#### 🗝️Key Concept

- The max circular subarray is basically:  
👉 `total array sum – min subarray sum`. Why?
- If you take the whole array and **remove the minimum chunk**, what’s left is the maximum _wrap-around_ subarray.
  So:
- `maxNormal = kadane(nums)`  // in case when there are all positive integers(normal array gives max)
- `maxCircular = totalSum – kadane(minSubarray)`
- Answer = max(maxNormal, maxCircular)

### 🧠Intuition/Approach

1. Run Kadane → get `maxNormal`.
2. If all elements are negative(maxNormal < 0) → return `maxNormal`.
3. Compute `totalSum` of array.
4. Run Kadane on inverted array → gives `-minSubarray`.
5. Compute `maxCircular = totalSum – minSubarray`.
6. Answer = `max(maxNormal, maxCircular)`.

### ❓Note

- Normal Kadane = “best line segment without crossing 0.”
- If it is < 0 , it means all elements are negative, in that case the inverse will give maxsubarray as full array, and total - max = 0, which doesn't make sense.
- Circular case = “best line segment that crosses 0.” Now: “Crossing 0” means you’re taking almost everything, except some middle chunk you cut out.
- That’s why it reduces to `total – minSubarray`.

### ⌛TC 
- Brute - O(n2/n3)
- Optimized - O(2n)

### 📦SC
- Brute - O(1)
- Optimized -O(1) 

---

### ✅Solution

```cpp
int kadane(vector<int> &nums){
        int currsum = nums[0], maxsum = nums[0];

        for(int i=1; i < nums.size(); i++){
            currsum = max(nums[i], currsum + nums[i]);
            maxsum = max(currsum, maxsum);
        }
         return maxsum;
    }

    int maxSubarraySumCircular(vector<int>& nums) {
      //step1 : find max subarray sum
      int normalkadane = kadane(nums);

      //All negative elements
      if(normalkadane < 0) return normalkadane;

        //step2 :Find total & Inverse the array and find the kadane
        int total = 0;

         for(int i = 0 ; i <nums.size(); i++){
                total += nums[i];
                nums[i] = -nums[i];
            }
        
            //step3 : flip the sign of the invkadane
            int minkadane = -kadane(nums);

        int circularkadane = total - minkadane;
        return max(normalkadane, circularkadane);
    }
```

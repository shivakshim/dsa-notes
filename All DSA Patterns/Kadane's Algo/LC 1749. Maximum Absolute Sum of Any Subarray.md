Return _the **maximum** absolute sum of any **(possibly empty)** subarray of_ `nums`.

---
#### ⌛Brute Force 

- Nested loops

#### 🗝️Key Concept

- Track both **max subarray sum** and **min subarray sum** in one Kadane pass → answer = max of those (absolute).

### 🧠Intuition/Approach

- Init `currmax = currmin = maxsum = nums[0]`.
- For each element: update `currmax = max(nums[i], currmax + nums[i])`.
- Update `currmin = min(nums[i], currmin + nums[i])`.
- Update `maxsum = max(maxsum, max(currmax, abs(currmin)))`.
- Return `maxsum`.

### ❓Note

- can be negavtive or positive we just want the most.

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(1)

---

### ✅Solution

```cpp
int maxAbsoluteSum(vector<int>& nums) {
        if(nums.size()==1) return abs(nums[0]);
        int currmax = nums[0], currmin = nums[0], maxsum = nums[0];
        for(int i=1 ; i<nums.size(); i++){
            currmax = max(nums[i], currmax + nums[i]);
            currmin = min(nums[i], currmin + nums[i]);
            maxsum = max(maxsum , max(currmax, abs(currmin)));
        }
        return maxsum;
    }
};
```

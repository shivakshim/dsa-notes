
- **Goal**: Return array where each element = product of all others, **no division**, O(n) time.
#### 🧠 Intuition (prefix && suffix products hashmaps) → **O(n) & NO Division**

  - i ➡️ left-to-right → store prefix product (product of all left elements).
  - j ⬅️ right-to-left → multiply with suffix product (product of all right elements).
- **Key**: No extra array (use output as prefix, temp for suffix). Handles zeros naturally.
- **Complexity**: O(n) time, O(1) extra space.

---
#### ✍️Note - 

- Since there's no prefix prod for i=0      &&      No suffix prod for j=n-1
     we initialize maps -> premap[0] = 1
				      sufmap[n-1] = 1

---

#### ✅Solution

```cpp
vector<int> productExceptSelf(vector<int>& nums) {
        int n = nums.size();
        vector<int> ans(n);
        unordered_map<int, int> premap;
        premap[0] = 1;
        unordered_map<int, int> sufmap;
        sufmap[n-1] = 1;
        int prefix = 1; int suffix = 1;
        
        for(int i=1, j=n-2; i<n && j>=0; i++ , j--){
            prefix *= nums[i-1];
            premap[i] = prefix;
            suffix *= nums[j+1];
            sufmap[j] = suffix;  
        }
        
        for(int i=0; i<n ; i++){
            ans[i] = premap[i] * sufmap[i];
        }
        
        return ans;
    }
```

## 🧠Intuition - 

- Variation - prefix sum +  sum freq hashmap
- We use a map to store presum->it's count.
            1. For every i we find the presum = presum += nums[i];
            2. for this presum we check if we have mp[presum - k] 
                  ->If yes - ans += count of this presum
                  ->no -- continue
- presum - previous_presum = k → previous_presum = presum - k
---

## ✍️Note - 

  1. mp[presum]++  should happen after the count is checked for presum-k, because we are going backwards              here.
  2. The map must be initialized with mp[0]=1; for the subarray starting from 0th index. For sum == k , diff = 0.
---

## ✅Solution - 

```cpp
int subarraySum(vector<int>& nums, int k) {
        int n = nums.size();
        unordered_map<int, int> mp;
        mp[0] = 1; //initialise 0
        int prefix = 0; int ans = 0;
        for(int i=0; i<n; i++)
        {
            prefix += nums[i];
            if(mp.find(prefix-k) != mp.end())
            ans += mp[prefix-k];
            mp[prefix]++;
        }
        return ans;
    }
```

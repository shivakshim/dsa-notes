🔗[leetcode-link](https://leetcode.com/problems/xor-queries-of-a-subarray/)

### 🧠Intuition - 

- Variation - XOR Prefix
- Key Idea - Find the prexor array -> return XOR[R-L] for each r, l.
            
### ✍️Note - 

  Prefix acts as inverse, so for XOR(L-R) to remove the overlap we don't subtract we do XOR for (l-1) :
            XOR[r-l] = XOR[r] ^ XOR[l-1]
            

### ✅Solution - 

```cpp
vector<int> xorQueries(vector<int>& arr, vector<vector<int>>& queries) {
        vector<int> prexor(arr.size(), 0);
        vector<int> ans;
        prexor[0] = arr[0];
        
        for(int i=1; i<arr.size(); i++){
          prexor[i] = prexor[i-1] ^ arr[i];
        }
        
        for(int i=0; i<queries.size(); i++){
          int left = queries[i][0];
          int right = queries[i][1];
          if(left == 0)  
            ans.push_back(prexor(right));
          else 
            ans.push_back(prexor[right] ^ prexor[left-1]);  //otherwise prexor[-1]=runtime e
        }
        return ans;
    }
```

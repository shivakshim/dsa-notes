## 🧠Intuition - 

- Variation - prefix Xor +  xor freq hashmap + xorIdxSum hashmap
- since we want a=b we should think -> the subarray [i , k] must have prexor = 0 ;
- So , whenever we find a repeating preXor at x and y , we know subarray(x+1, y) == 0 which is what we want.
- now since we want all possible j's present in this = y-(x+1) = y-x-1
- but this is for 1 subarray x->y we need for all that have same prexor ending at y , hence we take 2 maps :
    1. **freq** -> stores how many times a prexor has been found.
    2. **sumidx** - > stores the sum of all idx that have same prexor. 
    so our formula builds to -> 
      ***ans = freq[prexor] * (i-1) - sumidx[prexor]***
      

## ✍️Note - 

- Initialize -> freq[0] = 1; 
            sumIdx[0] = -1;
            

## ✅Solution - 

```cpp
int countTriplets(vector<int>& arr) {
        unordered_map<int, int> xorfreq;
        unordered_map<int, int> xoridxsum;
        xorfreq[0] = 1;
        xoridxsum[0] = -1;

        int prexor = 0; //since 0^n=n
        int ans = 0;

        for(int i=0; i<arr.size(); i++)
        {
           prexor ^= arr[i];
           if(xorfreq.count(prexor)){
            ans += xorfreq[prexor] * (i-1) - xoridxsum[prexor];
           }
           xorfreq[prexor]++;
           xoridxsum[prexor] += i;
        }
        return ans;
    }
```

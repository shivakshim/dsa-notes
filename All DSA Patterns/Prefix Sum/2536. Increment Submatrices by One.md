🔗[Problem Link](https://leetcode.com/problems/increment-submatrices-by-one/description/)
For each query [row1, col1, row2, col2] add 1 to the submatrix within this ***range*** and return the final matrix.

---
#### 🧠Intuition - Prefix sum + Difference Array
- Since we need to manipulate the range based on queries we use difference array, update required area by difference.
- We need to update total 4 indices :    1. starting idx(row1,col1) = +1
							    2. 1st row where addition stops(row2+1, col1) -= 1
							    3. 1st col where addition stops(row1, col2+1) -= 1
							    4. outer diagonal to addback 1 extra subtracted 1(row2+1, col2+1) += 1;
- Find the prefix sum matrix.

##### ❓Size of prefix matrix
- We can solve this by keeping size  = n, by simply putting checks for bounds at each update.
 
---

#### ✅Solution - 

```cpp
vector<vector<int>> rangeAddQueries(int n, vector<vector<int>>& queries) {
        vector<vector<int>> ans(n, vector<int>(n, 0));
        for(auto &t : queries){
          int row1 = t[0];
          int col1 = t[1];
          int row2 = t[2];
          int col2 = t[3];
          ans[row1][col1] += 1;
          if(row2 + 1 < n) ans[row2+1][col1] -= 1;
          if(col2 + 1 < n) ans[row1][col2+1] -= 1;
          if(row2 + 1 < n && col2 +1 < n) ans[row2+1][col2+1] += 1;
        }
        //calculate prefix matrix

        for(int i=0; i<n; i++){
            for(int j =0; j<n; j++){
               if(i > 0) ans[i][j] += ans[i-1][j];
               if(j > 0) ans[i][j] += ans[i][j-1];
               if(i > 0 && j > 0)    ans[i][j] -= ans[i-1][j-1];
            }
        }
        return ans;
    }

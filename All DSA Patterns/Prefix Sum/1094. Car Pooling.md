
🔗[leetcode-link](https://leetcode.com/problems/car-pooling/description/)
For array   `trips` where `trips[i] = [numPassengersi, fromi, toi]`, Return `true` _if it is possible to pick up and drop off all passengers for all the given trips, or_ `false` _otherwise_.

---
### 🧠Intuition - 

- Variation - prefix sum +  Difference Array
- We create a difference array of size max Distance till where the car is going, for each trip, we update the distances [l,r] by no. of passengers.
- Do Prefix sum on this difference array to determine no. pf passengers at any given distance, if it exceeds capacity return false else true.

---

### ✍️Note - 

  1. We have to make diff(n+1, 0), since distance starts from 1-n, we need nth idx too.

---

## ✅Solution - 

```cpp
bool carPooling(vector<vector<int>>& trips, int capacity) {
        int maxTo = 0;
        for (auto &t : trips) {
            maxTo = max(maxTo, t[2]);
        }
        
        //initilaise diff arr
        vector<int> diff(maxTo+1,0);
        for(auto &t : trips){
            diff[t[1]] += t[0];
            diff[t[2]] -= t[0];
        }

        //prefix of diff
        for(int i=1; i<maxTo; i++){
            diff[i] = diff[i-1] + diff[i];
        }
        
        if(*max_element(diff.begin(), diff.end()) <= capacity) return true;
        return false;

    }
```

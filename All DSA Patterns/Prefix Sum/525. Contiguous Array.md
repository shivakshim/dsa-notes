
🔗[Problem Link]([525. Contiguous Array](https://leetcode.com/problems/contiguous-array/))
***Find the longest subarray with equal number of 0s and 1s.***

---
#### 🧠Intuition - Prefix sum + hashmap
- Convert `0 → -1`.
- Then finding a subarray with equal 0s and 1s becomes finding a subarray whose **sum = 0**.

#### ✍️Prefix Sum Idea
- Maintain a running prefix sum.
- If the **same prefix sum** appears again, it means the numbers between are balanced (equal 0s and 1s).
- Use a hashmap to store the **first index of each prefix sum** → longest distance gives the answer.
- Initialize mp[0] = -1 , for case where presum == 0;

##### ❓Why not sliding window 
- Because the array isn’t guaranteed to have a **monotonic condition**—you don’t know when to shrink the window.
- Here, the subarray sum can swing both ways (0 → -1, 1 → +1), so removing elements from left doesn’t always bring you closer to a valid subarray.  
- ***You need to know the _first time_ each prefix sum occurred → hashmap, not window.**
 
---

#### ✅Solution - 

```cpp
int findMaxLength(vector<int>& nums) {
        unordered_map<int, int>mp;
        mp[0] = -1;
        int presum = 0; int ans = 0;
        for(int i=0; i<nums.size(); i++)
        {
            if(nums[i] == 0) presum += -1;
            else presum += 1;

            if(mp.count(presum))
             ans = max(ans, i - (mp[presum]));  //not  i - (mp[presum] + 1) because len = r-l+1
            else mp[presum] = i;
        }
        return ans;
    }
```



## 💡🧠 **FAANG-Tier MODIFIED BS — The Only 4 Patterns You Actually Need**

### 1️⃣ MODIFIED BINARY SEARCH (BSOA) — MUST DO

- Maximize min or minimize max.
- Monotonic nature of answer range : Condition not satisfied till a particular value, then satisfied by every vale after that.
- We do reverse engineering : Try to satisfy constraints based on the answer range. i.e., if my h < given hours, if my splits < given splits, my time < given time.

🔗[Problem link](https://leetcode.com/problems/remove-duplicates-from-sorted-list/description/)
Remove duplicates and return head.

---
#### ⌛Brute Force 

- Using set/nested loops.

#### 🗝️Key Concept

- Pointer clean up. Staggered pointers. q points to the next non-duplicate, p points to the 1st non duplicate. 
- p->next=q, till q!=nullptr.

### 🧠Intuition/Approach

- We keep p & q=p->next.
- at each step we check if q->val=p->val then keep moving q.
- If val changes we make p->next = q. Go on till q exists.

### ❓Note

- In the end we make p->next = nullptr, in case there were duplicates at the end & q reaches null then p keeps pointing the duplicates.

### ⌛TC 
- Brute - O(2n)
- Optimized - O(n)

### 📦SC
- Brute - O(n)
- Optimized - O(1)

---

### ✅Solution

```cpp
 ListNode* deleteDuplicates(ListNode* head) {
        if(!head || !head->next) return head;
        ListNode *p = head, *q = p->next;
        while(q){
            if(q->val != p->val) {
            p -> next = q;
            p = q;
            }
            q = q->next;
        }
        p->next = nullptr;
        return head;
    }
```

🔗[Problem link](https://leetcode.com/problems/rotate-list/description/)
Rotate list to right.

---

#### 🗝️Key Concept

- Fast & slow pointers to find out the new head(slow) and join the end to old head(fast). 
- Similar to [[19. Remove Nth Node From End of List]]

### 🧠Intuition/Approach

- We use fast & slow pointer (relative by gap).
- fast move forward by k elements so that when slow & fast both move together & fast reaches the end - slow lands 1 element before the supposed end of the rotated list.
- then we simply make fast->next =  head,  slow->next = head, & slow->next = nullptr.

### ❓Note

- Handle the edge case for !head || !head->next || k=0 at start.

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n)

### 📦SC
- Brute - O(n)
- Optimized - O(1)

---

### ✅Solution

```cpp
ListNode* rotateRight(ListNode* head, int k) {
        if(!head || !head->next || k==0) return head;
        ListNode *fast = head;
        ListNode *slow = head;
        for(int i = 1; i<=k ;i++){
            fast = fast->next ? fast->next : head;
        }
        while(fast->next){
            slow = slow->next;
            fast = fast->next;
        }
        fast->next = head;
        head = slow->next;
        slow->next = nullptr;
        return head;
    }
```

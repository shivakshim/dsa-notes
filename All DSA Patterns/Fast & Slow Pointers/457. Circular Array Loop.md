Detect cycle in a circular array where nums[i] gives the jumps to be taken from i'th place.

---
#### ⌛Brute Force 

- 

#### 🗝️Key Concept

- We need to look at the array as a linked list, where the next pointer is defined by nextIdx = (arr[i]+ i ) % n. Then we simply need to look for cycle with fast and slow pointers.

### 🧠Intuition/Approach

- STEP 1 : We check if the given node/path is visited nums[i] == 0 continue;
- STEP 2 : Define the direction as the direction of the node we are standing at. And define slow=fast=i.
- STEP 3 : Get nextslow & nextfast by a helper function. check if they are vis || !dir -> break;
- STEP 4 : If fast == slow -> check if its a self loop by if(slow->next) == slow break;
- STEP 5 : Update the path as visited.

### ❓Note

- actual array is of i, which is circular, nums[i] juts defines the jumps on it.

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n)

### 📦SC
- Brute - O(n)
- Optimized - O(1)

---

### ✅Solution

 ```cpp
int next(vector<int> &nums, int i){
        int n = nums.size();
        return ((nums[i] + i) % n + n ) % n;
    }
    bool circularArrayLoop(vector<int>& nums) {
        int n = nums.size();
        for(int i = 0; i<n ; i++){
            //1️. check if visited
            if(nums[i] == 0) continue;

            //2.define dir, slow, fast
            bool dir = nums[i] > 0;
            int slow = i, fast = i;

            while(true){
                //3.Move slow & fast pointers & check for dir & vis
                int nextslow = next(nums, slow);
                if(nums[nextslow] == 0 || nums[nextslow] > 0 != dir) break;

                int nextfast = next(nums, fast);
                if(nums[nextfast] == 0 || nums[nextfast] > 0 != dir) break;

                nextfast = next(nums, nextfast);
                if(nums[nextfast] == 0 || nums[nextfast] > 0 != dir) break;

                fast = nextfast;
                slow = nextslow;

                if(fast == slow){
                    //4.check for self loop
                    if(next(nums, slow) == slow) break;
                    return true;
                }
            }
            //5. Mark path as visited
            int idx = i;
            while(nums[idx] != 0 && nums[idx] > 0 == dir) {
                int nextidx = next(nums, idx);
                nums[idx] = 0;
                idx = nextidx;
            }

        }
        return false;
    }
 ```

🔗[Problem link]()
Remove nth node from the end of the LinkedList in a single pass (without calculating the length first).

---
#### ⌛Brute Force 

######       🛑 2-pass method 
1. Traverse list once → count total length `len`.
2. Compute `pos = len - n` (0-based).
3. Traverse again → delete node at `pos`. 👉 Works, but **two passes**.

#### 🗝️Key Concept

- Maintain a **gap** between two pointers so the “slow” one lands exactly where you want when “fast” hits the end.
- It is essentially fast & slow pointers just in terms of gap not speed.
- The “1 pass” trick = **make fast & slow maintain a fixed gap (n nodes)**.  When fast reaches the end → slow is at the perfect spot to cut.

### 🧠Intuition/Approach

-  Step 1: Create a gap -
   Move `fast` pointer **n steps ahead** of `slow`. Now between `fast` and `slow`, there are exactly **n nodes**.
-  Step 2: March together -  Move both `fast` and `slow` **1 step at a time** until `fast` reaches the last node (end).           Think: because of the fixed gap, by the time `fast` is at the end, `slow` will be **n nodes behind the end**.
- Step 3: Delete - Just do `slow->next = slow->next->next`.

### ❓Note

- If `fast` starts from head and moves `n` steps, it might land at `nullptr`.
- That’s why we use a **dummy node** at the start → so `slow` is always one step before the node to delete, even if it’s the head.
- We return dummy->next, not head as head might be deleted.

### ⌛TC 
- Brute - O(2n)
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(1)

---

### ✅Solution

```cpp
 ListNode* removeNthFromEnd(ListNode* head, int n) {
        ListNode *dummy = new ListNode;
        dummy -> next =  head;
        ListNode *slow = dummy;
        ListNode *fast = dummy;
        //move fast n steps ahead of slow
        for(int i = 1; i <= n; i++){
            fast = fast -> next;
        }
        while(fast->next){
            cout<<"enetered while";
            slow = slow -> next;
            fast = fast -> next;
        }
        //now slow is just before the nth node from the end
        slow->next = slow->next->next;
        return dummy->next;
    }
```

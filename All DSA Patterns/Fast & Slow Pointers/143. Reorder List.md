Reorder the list to be on the following form: L0 → Ln → L1 → Ln - 1 → L2 → Ln - 2 → …

---
#### 🗝️Key Concept

- Find mid(Floyd's algo) + reverse list2 + merge lists.

### 🧠Intuition/Approach

- Step1 : With [[🐢Floyd’s Cycle Detection Algorithm]] , find the mid of the linkedlist (slow ptr).
- Step2 : Keep head2 of 2nd list at slow->next, disconnect the 2 lists -> head - mid(slow) | head2(slow -> next) - end.
- Step3 : Merge list1 and list2.

### ❓Note

- To merge the lists - while(head2).
- keep 2 pointers at head1->next, head2->next .
- move accordingly.

### ⌛TC 
- Brute - 
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(1)

---

### ✅Solution

```cpp
 void reorderList(ListNode* head) {
        ///step1: Find the mid element - Floyd's algo
        ListNode *slow = head;
        ListNode *fast = head;
        while(fast && fast->next){
            slow = slow->next;
            fast= fast->next->next;
        }

        //step2 : slow is at mid, Reverse list after mid
        ListNode *p = slow->next;
        slow->next = nullptr;  //disconnect the lists
        ListNode *q = nullptr;
        ListNode *r = nullptr;
        while(p){
            r = q;
            q = p;
            p = p->next;
            q->next = r;   //q points to head of list2
        }

        //step3 : Merge the 2 lists
        ListNode *h1 = head;
        ListNode *h2 = q;
        while(h2){
            ListNode *t1 = h1->next;
            ListNode *t2 = h2->next;
            h1 -> next = h2;
            h2 -> next = t1;
            h1 = t1;
            h2 = t2;
        }
    }
```

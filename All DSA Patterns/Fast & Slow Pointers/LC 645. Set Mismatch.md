🔗[Problem link](https://leetcode.com/problems/set-mismatch/description/)
In an array [1-n] find the duplicate and the missing.
(*💡**NOTE** : This is not a fast & slow pointer question, it is juts a follow up of [[287. Find the Duplicate Number]])

---
#### ⌛Brute Force 

- use set/ sort array.

#### 🗝️Key Concept

- **Cyclic sort**: swaps each number into its “correct chair” (`num` → index `num-1`).

### 🧠Intuition/Approach

-  **Cyclic sort**: swaps each number into its “correct chair” (`num` → index `num-1`).
- **Duplicate detection**: if a number’s chair is already taken, it’s the duplicate.
- **Missing detection**: the index whose chair didn’t get its correct number.

### ❓Note

-  `for i = 0 to n-1` → iterates through the array.
- `while nums[i] != nums[nums[i]-1]` → keeps swapping the number into its correct position until it’s done.

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n)

### 📦SC
- Brute - O(n)
- Optimized - O(1)

---

### ✅Solution

```cpp
 vector<int> findErrorNums(vector<int>& nums) {
        int n = nums.size();
        for(int i=0; i<n; i++){
            while(nums[i] != nums[nums[i]-1]){
                swap(nums[i], nums[nums[i]-1]);
            }
        }
        int duplicate = -1;
        int missing = -1;
        int i = 0;
        while(nums[i] == i+1){
           i++;
            }
        return {nums[i], i+1};
    }
```

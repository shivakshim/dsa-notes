🔗[Problem link]()
Remove all elements that have duplicates.

---

#### 🗝️Key Concept

- 💡***NOTE  : IN ANY LINKEDLIST PROBLEM WHERE WE MIGHT HAVE TO DELETE THE HEAD, USE A DUMMY NODE.
-  Using a nested loop to only point prev unique to next if its not duplicate.

### 🧠Intuition/Approach

- Create a dummy before head, set `p=head`, `q=head->next`, `prev=dummy`.
- While `q` exists:
    - If `p->val == q->val`, move `q` forward until value changes.
    - Else link `p` to result (`prev->next = p; prev=p`).
    - Update `p=q`, `q=q?->next`.
- After loop, link last candidate (`prev->next = p`).
- Return `dummy->next`.

### ❓Note

- In the end we point prev to p since it might get missed is q already moved our of loop.

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n2)

### 📦SC
- Brute - O(n)
- Optimized - O(1)

---

### ✅Solution

```cpp
 ListNode* deleteDuplicates(ListNode* head) {
        if(!head || !head->next) return head;
        ListNode *dummy = new ListNode(0, head);
        ListNode *p = head, *q = head->next, *prev = dummy;
        while(q){
            if(p->val == q->val) {
            while(q && p->val == q->val) q=q->next;
            }
            else{
                prev->next = p;
                prev = p;
            }
            p = q;
            if(q) q = q->next;
        }
        prev -> next = p;
        return dummy->next;
    }
```

🔗[Problem link](https://leetcode.com/problems/odd-even-linked-list/description/)
All odd indexes first then all even indexes.

---

#### 🗝️Key Concept

- Keep updating the list with 2 pointers till the even's pointer is present.

### 🧠Intuition/Approach

- Set odd = head, even = head->next, store evenHead.
- While even and even->next exist →
    - Link odd->next to even->next, move odd forward.
    - Link even->next to odd->next, move even forward.
- Join odd->next = evenHead.
- Return head.

### ❓Note

- We only need to move till odd is there or odds next is there because p must never go to null, ad p = q-> next.

### ⌛TC 
- Brute - O(2n)
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(1)

---

### ✅Solution

```cpp
ListNode* oddEvenList(ListNode* head) {
        if(!head || !head->next) return head;
        ListNode *p = head, *q = head->next, *h2 = q;
        
        while(q && q->next){
            p->next = q->next;
            p = q->next;
            q->next = p->next;
            q = p->next;
        }
        
        p->next = h2;
        return head;
    }
```

🔗[Problem link](https://leetcode.com/problems/jump-game-ii/submissions/1809338955/)
Given `nums[i]` = max jump length at index `i`, find **min jumps** to reach last index.

---
#### ⌛Brute Force 

1. Find all paths.
2. Loop all possible indices in range.

#### 🗝️Key Concept

- Greedily keep finding the global maxreach for every i, and whenever the i=endpoint of the curr range, take a jump and reset endpoint = maxreach.
- maxreach = how far we can go till now, endpoint = till which index are included in current range.

### 🧠Intuition/Approach

- `maxreach = 0`, `end = 0`, `jumps = 0`
- Loop `i` from 0 to `n-2`:
    - `maxreach = max(maxreach, i + nums[i])`
    - If `i == end`: `jumps++`, `end = maxreach
- Return `jumps`

### ❓Note

- We don't need loop for the range, we simply keep a pointer to track the end the loop.

### ⌛TC 
- Brute - O(n2)
- Optimized - O(n)

### 📦SC
- Brute - O(1)
- Optimized - O(1)

---

### ✅Solution

```cpp
 int jump(vector<int>& nums) {
        int jumps = 0, maxreach = 0, steps = 0;
        for(int i=0; i<nums.size()-1; i++){
            //find maxreach till now
            maxreach = max(maxreach, nums[i] + i);
            
            //end of curr range
            if(i == steps){
                jumps++;
                steps = maxreach;
            }
        }
        return jumps;
    }
```

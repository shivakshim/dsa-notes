🔗[Problem link](https://leetcode.com/problems/candy/description/)
There are `n` children standing in a line. Each child is assigned a rating value given in the integer array `ratings`.
You are giving candies to these children subjected to the following requirements:
- Each child must have at least one candy.
- Children with a higher rating get more candies than their neighbors.
Return _the minimum number of candies you need to have to distribute the candies to the children_.

---

#### 🗝️Key Concept

- ##### rule of thumb:
> 🔹 If "neighbor" = next index → **scan**.  
> 🔹 If "neighbor" = next in sorted order by start/end → **sort + scan**.

- First pass ensures each kid has more candies than the **left** lower-rated kid.
- Second pass ensures the same for the **right** one.

### 🧠Intuition/Approach

1️⃣ **Init:** Create `candies` array of size `n`, all 1s (everyone gets 1 candy).  
2️⃣ **Left → Right pass:**  
If `ratings[i] > ratings[i-1]`, then `candies[i] = candies[i-1] + 1`.  
3️⃣ **Right → Left pass:**  
If `ratings[i] > ratings[i+1]`, then `candies[i] = max(candies[i], candies[i+1] + 1)`.  
4️⃣ **Sum up:** Return total `candies` using `accumulate`.

### ❓Note

- 🧠Idea - Each kid should have more candies than a lower-rated neighbor — so check both sides once."

### ⌛TC 
- Brute - O(N2)
- Optimized - O(2n)

### 📦SC
- Brute - O(1)
- Optimized - O(n)

---

### ✅Solution

```cpp
 int candy(vector<int>& ratings) {
        int n = ratings.size();
        vector<int> candies(n, 1);

        //Pass 1  : Left to right
        for(int i = 1; i < n ; i++){
            if(ratings[i] > ratings[i-1])
            candies[i] = candies[i-1] + 1;
        }

        //Pass 2 : Right to left
        for(int i = n-2; i >= 0; i--){
            if(ratings[i] > ratings[i+1])
            candies[i] = max(candies[i], candies[i+1]+1);
        }
        
        return accumulate(candies.begin(), candies.end(), 0);
    }
```
